# InstantInsanitySolutionFinder
Script to find a solution to the Instant Insanity puzzle game

Finds solution ([front, right, back, left, top, bottom])

```
[
    ["W", "R", "G", "B", "R", "R"],
    ["B", "G", "W", "R", "R", "W"],
    ["R", "B", "B", "W", "G", "W"],
    ["G", "W", "R", "G", "B", "B"]
]
```
